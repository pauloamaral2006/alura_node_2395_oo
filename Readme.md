## 📘Ementa

### JavaScript: programação orientada a objetos

- Aprender os conceitos principais da orientação a objetos
- Entender como o JavaScript lida com a OO através de funções e protótipos
- Ver os princípios de classe, objeto e herança
- Descubrir o que é o encapsulamento e por que é importante
- Entender o polimorfismo com exemplos
- Analisar seu código de acordo com os padrões do SOLID
